面 向教学科研的统一大模型 API 平台

三步获取密钥，
接入大语言模型

为方便统一管理大模型 API 与配额，南京大学软件学院、智能软件与工程学院 基于开源软件 New API 部 署了本平台，为同学们分配独立的 API 密钥，支持调用多种主流大语言模型，用于教学与科研实验。

1 API 端点地址 — 所有同学共用统一 Base URL：https://njusehub.info/v1
2 API 密钥 — 登录控制台后在「API 密钥管理」页面创建，用于身份认证与用量统计
3 模型 ID — 在「模型广场」页面浏览可用模型并记录 ID，如 glm-5.2，需与客户端配置一致
⚠ 务必保管好你的 API 密钥，切记不要将其上传至 GitHub 等外部平台。
